# 更新总结 - 按文件名序号排序

## 🎯 本次更新目标
修复 Obsidian 导入的笔记在系列页面中的排序问题，实现按文件名序号排序功能。

## ✅ 实现的功能

### 1. 按文件名序号排序
- ✅ 支持数字序号：`1. 标题.md`、`2. 标题.md`、`10. 标题.md`
- ✅ 支持字母序号：`A. 标题.md`、`B. 标题.md`
- ✅ 数字序号优先于字母序号
- ✅ 无序号文件排在最后
- ✅ 数字按数值大小排序（正确处理 1, 2, 10 的顺序）

### 2. 序号提取逻辑
- ✅ 从文件名中提取前缀序号
- ✅ 支持分隔符 `-` 和 `.`
- ✅ 兼容多种命名格式

## 🔍 支持的文件名格式

| 格式 | 序号 | 排序 |
|------|------|------|
| `1. 标题.md` | 1 | 第 1 位 |
| `2. 标题.md` | 2 | 第 2 位 |
| `10. 标题.md` | 10 | 第 10 位 |
| `A. 标题.md` | A | 数字序号后 |
| `B. 标题.md` | B | 数字序号后 |
| `标题.md` | 无 | 最后 |

## 🎯 适用场景

1. **Obsidian 笔记** - 带序号的笔记文件
2. **教程系列** - 按章节排列的文章
3. **课程资料** - 按课序排列的文档
4. **技术文档** - 按章节编号排列的内容

## ⚠️ 注意事项

- 仅影响**系列页面**的排序
- 主页文章仍按日期排序
- 文件名必须以 `序号.` 或 `序号-` 开头
- 序号分隔符只能是 `-` 或 `.`

## 🚀 使用方式

无需配置，自动生效！

1. 导入 Obsidian 笔记：
   ```bash
   pnpm import-obsidian ~/MyNotes
   ```

2. 访问系列页面：
   ```
   /series/你的系列名称/
   ```

3. 文档自动按文件名序号排序！

## 🔧 技术实现

使用正则表达式提取序号：
```typescript
const filename = post.slug.split("/").pop() || post.slug;
const match = filename.match(/^(\d+|[A-Za-z])[-.]/);
```

排序逻辑：
- 数字序号：数值比较（1 < 2 < 10）
- 字母序号：字符串比较（A < B < C）
- 数字优先于字母
- 无序号：Infinity（排在最后）
